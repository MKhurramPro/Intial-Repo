# My First HTML Project

## 📋 Project Description

This is a beginner-level HTML learning project designed to demonstrate fundamental web development concepts and HTML5 semantic markup. The project consists of multiple interconnected HTML pages that showcase various HTML elements and best practices.

## 📁 Project Structure

```
Intial-Repo/
├── index.html           (Main landing page)
├── Contact us.html      (Contact/data showcase page)
├── GoodBye.html         (Semantic HTML elements page)
├── Folder/
│   └── Hello.html       (Simple greeting page)
│   └── Image.jpg        (Referenced image file)
└── README.md            (This file)
```

## 🎯 Project Overview

### Pages

#### 1. **index.html** - Main Landing Page
- **Purpose:** Demonstrates basic HTML elements and text formatting
- **Key Features:**
  - Multiple heading levels (h1-h4)
  - Paragraph text with Lorem ipsum
  - External hyperlinks
  - Image embedding
  - Text formatting (bold, italic, underline)
  - Big and small text elements
  - Subscript and superscript notation (H₂O, 9²=81)
  - Pre-formatted text
  - Horizontal rules
  - Navigation links to other pages

#### 2. **Contact us.html** - Contact & Data Showcase
- **Purpose:** Demonstrates lists and table structures
- **Key Features:**
  - Unordered lists (ul/li)
  - Ordered lists (ol/li)
  - Nested lists
  - HTML tables with headers and body
  - Table captions
  - Student data presentation

#### 3. **GoodBye.html** - Semantic HTML Page
- **Purpose:** Showcases semantic HTML5 elements
- **Key Features:**
  - Semantic header element
  - Main content wrapper
  - Section elements for content grouping
  - Article element
  - Aside element (for advertisements)
  - Footer element
  - Embedded images and links
  - Div and span elements

#### 4. **Folder/Hello.html** - Simple Greeting Page
- **Purpose:** Basic page with minimal HTML structure
- **Key Features:**
  - Simple h1 heading
  - Demonstrates subdirectory organization

## 🎓 Learning Outcomes

This project covers the following HTML concepts:

### Basic Elements
- Document structure (DOCTYPE, html, head, body)
- Meta tags (charset, viewport)
- Headings (h1-h6)
- Paragraphs
- Line breaks and horizontal rules

### Text Formatting
- Bold text (`<b>`, `<strong>`)
- Italic text (`<i>`, `<em>`)
- Underline (`<u>`)
- Big and small text
- Superscript (`<sup>`) and subscript (`<sub>`)
- Pre-formatted text (`<pre>`)

### Links & Media
- Hyperlinks (`<a>`)
- Image embedding (`<img>`)
- External and internal links
- Cross-page navigation

### Lists
- Unordered lists (`<ul>`)
- Ordered lists (`<ol>`)
- List items (`<li>`)
- Nested lists

### Tables
- Table structure (`<table>`, `<thead>`, `<tbody>`)
- Table headers (`<th>`)
- Table data cells (`<td>`)
- Row and column spanning
- Table captions

### Semantic HTML5
- Header (`<header>`)
- Main (`<main>`)
- Section (`<section>`)
- Article (`<article>`)
- Aside (`<aside>`)
- Footer (`<footer>`)
- Div and span elements

## 🚀 How to Use

1. **Open in Browser:** Simply open `index.html` in any modern web browser
2. **Navigate:** Use the links provided in each page to navigate between pages
3. **View Source:** Right-click on any page and select "View Page Source" to inspect the HTML code
4. **Learn:** Study each page to understand how different HTML elements work
